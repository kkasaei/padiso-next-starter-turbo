# SearchFI.ai Monorepo

This repository contains the SearchFI.ai applications and shared packages.

## Apps

- `apps/web`: Main SearchFI.ai web app (Next.js).
- `apps/embed`: Embeddable widget/client used by partners and integrations.

## Packages

- `packages/ui`: Shared UI components, fragments, and styles.
- `packages/common`: Shared constants and utilities used across apps.
- `packages/analytics`: Analytics helpers and events.
- `packages/react-providers`: Shared React providers and client setup.
- `packages/typescript-config`: Base TypeScript configs for the monorepo.
- `packages/eslint-config`: Shared ESLint configuration.

## Technologies

- Turborepo for monorepo orchestration and task running.
- Next.js for the web application framework.
- React for UI development.
- TypeScript for type-safe code.
- Tailwind CSS for styling.
- pnpm for package management.
