SearchFit's Technical Audit feature provides a comprehensive analysis of your website's technical SEO health, identifying issues that affect both search engine and AI platform visibility.

## Running an Audit

### Quick Audit

For a fast overview:

1. Go to **Technical Audit** → **Quick Scan**
2. Enter your domain
3. Receive results in under 2 minutes

Quick audits check:
- Critical errors
- Page speed basics
- Mobile friendliness
- Core Web Vitals

### Full Audit

For comprehensive analysis:

1. Go to **Technical Audit** → **Full Audit**
2. Configure crawl settings
3. Set crawl depth and page limits
4. Start the audit

Full audits include:
- Complete site crawl
- All technical SEO factors
- Detailed recommendations
- Priority scoring

## Audit Categories

### Crawlability

Ensures search engines and AI can access your content:

- Robots.txt analysis
- XML sitemap validation
- Crawl errors and redirects
- Orphan page detection

### Indexability

Checks that pages can be indexed:

- Meta robots directives
- Canonical tags
- Duplicate content
- Index status

### Performance

Measures site speed and performance:

- Core Web Vitals (LCP, FID, CLS)
- Page load times
- Resource optimization
- Mobile performance

### On-Page SEO

Analyzes page-level optimization:

- Title tags and meta descriptions
- Heading structure (H1-H6)
- Image optimization
- Internal linking

### Structured Data

Validates structured markup:

- Schema.org implementation
- Rich snippet eligibility
- JSON-LD validation
- FAQ and HowTo schemas

### Security

Checks security best practices:

- HTTPS implementation
- Mixed content issues
- Security headers
- SSL certificate status

## Issue Prioritization

Issues are categorized by severity:

| Level | Description | Action |
|-------|-------------|--------|
| Critical | Blocking issues that prevent indexing | Fix immediately |
| High | Significant impact on visibility | Fix within a week |
| Medium | Moderate impact on performance | Fix within a month |
| Low | Minor improvements | Fix when possible |

## AEO-Specific Checks

SearchFit includes AI visibility-specific audits:

### Entity Clarity
- Brand entity definition
- Organization schema
- About page optimization

### Answer Optimization
- FAQ implementation
- Question-based content structure
- Direct answer formatting

### Citation Readiness
- Source credibility signals
- Author information
- Publication dates

## Comparison Reports

Compare audits over time:

1. Run regular audits
2. Go to **Technical Audit** → **Compare**
3. Select two audit dates
4. View improvements and regressions

## Automated Monitoring

Set up continuous monitoring:

1. Go to **Technical Audit** → **Monitoring**
2. Enable automated weekly audits
3. Configure alert thresholds
4. Receive notifications for new issues

## Exporting Results

Export audit data in multiple formats:

- **PDF Report** - Formatted for stakeholders
- **CSV Data** - Raw issue data
- **Task List** - Import to project management tools

## Best Practices

1. **Audit monthly** - Run full audits at least monthly
2. **Fix critical first** - Prioritize blocking issues
3. **Track progress** - Compare audits over time
4. **Verify fixes** - Re-audit after making changes
5. **Monitor continuously** - Set up automated monitoring
