SearchFit Analytics provides comprehensive insights into your performance across both traditional search and AI platforms.

## Dashboard Overview

Your analytics dashboard shows:

- **Overall Visibility Score** - Combined SEO + AEO performance
- **Platform Breakdown** - Performance by platform
- **Trend Charts** - Historical performance data
- **Quick Insights** - AI-generated recommendations

## Key Metrics

### Visibility Metrics

| Metric | Description |
|--------|-------------|
| AEO Score | Your AI visibility rating (0-100) |
| Mention Count | Total mentions across AI platforms |
| Sentiment Score | Positive/neutral/negative analysis |
| Accuracy Rate | How often AI information is correct |

### SEO Metrics

| Metric | Description |
|--------|-------------|
| Organic Traffic | Search engine visitors |
| Keyword Rankings | Position tracking |
| Backlink Count | Total referring domains |
| Domain Authority | Overall domain strength |

### Combined Insights

SearchFit correlates SEO and AEO data to provide unique insights:

- How SEO improvements affect AI visibility
- Which content performs best across both channels
- Where to focus optimization efforts
