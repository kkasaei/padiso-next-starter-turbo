## Setting Up Integrations

### OAuth Connections

For most integrations:

1. Go to **Settings** → **Integrations**
2. Find the integration you want
3. Click **Connect**
4. Authorize SearchFit in the popup
5. Configure sync settings

### API Key Connections

Some integrations require API keys:

1. Get your API key from the service
2. Go to **Settings** → **Integrations**
3. Click **Connect** on the integration
4. Enter your API key
5. Test the connection

### Webhook Integrations

For custom integrations:

1. Go to **Settings** → **Integrations** → **Webhooks**
2. Create a new webhook
3. Copy the webhook URL
4. Configure in your external service
5. Test with a sample payload

## Data Sync Options

Configure how data syncs:

### Sync Frequency
- **Real-time** - Immediate updates
- **Hourly** - Every hour
- **Daily** - Once per day
- **Manual** - On-demand only

### Data Direction
- **Import only** - Pull data into SearchFit
- **Export only** - Push data out
- **Bidirectional** - Two-way sync

## API Access

Build custom integrations with our API:

### Authentication
```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.searchfit.com/v1/reports
```

### Endpoints

| Endpoint | Description |
|----------|-------------|
| `/reports` | Access AEO reports |

## Zapier Integration

Connect SearchFit to 5,000+ apps via Zapier:

1. Go to **Settings** → **Integrations** → **Zapier**
2. Click **Connect to Zapier**
3. Create Zaps with SearchFit triggers and actions

### Available Triggers
- New report generated
- Visibility score change
- New mention detected

### Available Actions
- Generate report
- Add domain
- Export data

## Troubleshooting

### Connection Issues

If an integration isn't working:

1. Check the integration status in Settings
2. Verify API keys are current
3. Reauthorize OAuth connections
4. Check the service's status page

### Data Sync Delays

If data isn't syncing:

1. Check sync frequency settings
2. Verify data permissions
3. Review error logs in Settings
4. Contact support if issues persist

## Best Practices

1. **Connect essential tools first** - Start with Analytics and Search Console
2. **Review permissions** - Only grant necessary access
3. **Monitor sync status** - Check integrations regularly
4. **Keep credentials secure** - Rotate API keys periodically
5. **Use webhooks wisely** - Don't overload external services
