SearchFit generates professional PDF reports that you can download, share with stakeholders, and use for presentations.

## Generating a PDF Report

### From Dashboard

1. Navigate to any report in your dashboard
2. Click the **Download PDF** button
3. Wait for PDF generation (usually 10-30 seconds)
4. Download starts automatically

### From Report Page

1. View your full AEO report
2. Click **Export** → **Download PDF**
3. Select sections to include (optional)
4. Generate and download

## PDF Contents

Your PDF report includes:

### Cover Page
- Report title and date
- Domain analyzed
- Overall AEO score
- SearchFit branding

### Executive Summary
- Key findings overview
- Score breakdown by platform
- Top 3 recommendations
- Comparison to previous report

### Detailed Analysis
- Platform-by-platform breakdown
- Mention analysis
- Sentiment overview
- Accuracy assessment

### Recommendations
- Prioritized action items
- Impact assessment
- Implementation guidance
- Resource requirements

### Appendix
- Raw data tables
- Methodology explanation
- Glossary of terms

## Customizing PDFs

### Branding Options

For Pro and Enterprise plans:

1. Go to **Settings** → **Branding**
2. Upload your logo
3. Set brand colors
4. Choose header/footer text
5. Save branding preferences

### Section Selection

Choose which sections to include:

- Toggle sections on/off before generating
- Reorder sections as needed
- Add custom notes or context
- Include or exclude competitor data

## Sharing Reports

### Direct Download

Download and share manually via:
- Email attachment
- Cloud storage (Google Drive, Dropbox)
- Team messaging (Slack, Teams)

### Shareable Link

Create a shareable link:

1. Generate your PDF report
2. Click **Share** → **Create Link**
3. Set expiration (optional)
4. Add password protection (optional)
5. Copy and share the link

### Scheduled Delivery

Automatically send reports:

1. Go to **Reports** → **Schedule**
2. Enable PDF delivery
3. Add recipient email addresses
4. Set delivery frequency
5. Customize email message

## Report History

Access previously generated PDFs:

1. Go to **Reports** → **History**
2. View all generated reports
3. Re-download any past report
4. Compare reports over time

Generate PDFs programmatically:

```bash
curl -X POST https://api.searchfit.com/v1/reports/pdf \
  -H "Authorization: Bearer YOUR_API_KEY" \

```

## Troubleshooting

### PDF Won't Generate

If PDF generation fails:

1. Check your internet connection
2. Ensure the report has completed
3. Try a different browser
4. Clear browser cache
5. Contact support if issues persist

### PDF Looks Wrong

If formatting is off:

1. Use latest browser version
2. Download fresh copy
3. Check PDF viewer settings
4. Report issue with screenshot

### Download Issues

If download doesn't start:

1. Check browser download settings
2. Disable ad blockers temporarily
3. Try right-click → Save As
4. Use a different browser
