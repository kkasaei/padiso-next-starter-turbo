## The Stack Architecture

### Layer 1: Data Foundation — SearchFit.ai
SearchFit.ai connects to your Shopify store and aggregates:
- Product catalog data (titles, descriptions, attributes, images)
- Search Console performance data (impressions, clicks, rankings)
- GA4 traffic and conversion data
- Competitor intelligence

This data foundation informs every AI operation in the stack.

### Layer 2: Content Generation — GPT-5.2 + Claude 4.6 Opus
Using the data from Layer 1:
- **GPT-5.2** generates product descriptions, meta tags, and schema markup
- **Claude 4.6 Opus** generates collection page content, blog posts, and buying guides
- **Claude 4.5 Sonnet** handles volume production for lower-priority content
- Content is generated with real keyword data and performance context

### Layer 3: Technical Implementation — Cursor
Cursor powered by Claude 4.6 Opus handles:
- Shopify Liquid theme modifications
- Core Web Vitals optimization
- Schema markup deployment
- SEO automation scripts
- Migration tasks

### Layer 4: Monitoring and Optimization — SearchFit.ai
After content is live:
- Track keyword ranking changes per page
- Monitor revenue attribution from organic traffic
- Identify underperforming pages for re-optimization
- Automate iterative improvements based on performance data
