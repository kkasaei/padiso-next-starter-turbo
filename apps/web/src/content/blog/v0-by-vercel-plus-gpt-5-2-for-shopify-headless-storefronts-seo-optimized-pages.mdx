## SEO Checklist for Headless Shopify Pages

Every page you build with V0 + GPT-5.2 should include:

- [ ] Unique meta title (under 60 characters)
- [ ] Unique meta description (under 155 characters)
- [ ] Proper heading hierarchy (single H1, logical H2/H3 structure)
- [ ] Product JSON-LD schema
- [ ] BreadcrumbList schema
- [ ] FAQPage schema (if FAQ section exists)
- [ ] Canonical URL
- [ ] Open Graph tags for social sharing
- [ ] Sitemap inclusion
- [ ] Internal links to related pages
- [ ] Image alt text on all images
- [ ] Mobile-responsive layout
