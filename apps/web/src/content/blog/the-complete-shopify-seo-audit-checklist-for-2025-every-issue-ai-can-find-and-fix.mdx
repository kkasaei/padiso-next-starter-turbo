A proper Shopify SEO audit is the foundation of any optimisation strategy. But most audit guides are outdated, generic, or don't account for Shopify-specific issues. This is the complete checklist for 2025—covering every technical, content, and structural issue that impacts your rankings, with notes on how AI tools can detect and fix each one.

‍

## Part 1: Technical SEO Audit

### Crawlability
- [ ] Robots.txt isn't blocking important pages
- [ ] XML sitemap is submitted and processing in Search Console
- [ ] All important pages return 200 status codes
- [ ] No redirect chains (A→B→C should be A→C)
- [ ] No redirect loops
- [ ] Crawl budget isn't wasted on low-value pages

### Indexation
- [ ] All product pages are indexed
- [ ] All collection pages are indexed
- [ ] Blog posts are indexed
- [ ] No important pages have accidental noindex tags
- [ ] Duplicate pages are properly canonicalized
- [ ] Variant URLs aren't creating duplicate indexation

### Page Speed (Core Web Vitals)
- [ ] LCP under 2.5 seconds on mobile
- [ ] CLS under 0.1
- [ ] INP under 200ms
- [ ] Images are optimized (WebP, proper sizing, lazy loading)
- [ ] No render-blocking JavaScript or CSS
- [ ] Third-party scripts are deferred

### Mobile
- [ ] All pages pass Google's Mobile-Friendly Test
- [ ] Touch targets are at least 48px
- [ ] No horizontal scrolling issues
- [ ] Text is readable without zooming
- [ ] Mobile navigation works correctly

### HTTPS and Security
- [ ] SSL certificate is active and valid
- [ ] No mixed content warnings
- [ ] All internal links use HTTPS
- [ ] HSTS headers are configured

## Part 2: On-Page SEO Audit

### Meta Tags
- [ ] Every page has a unique meta title (under 60 characters)
- [ ] Every page has a unique meta description (under 155 characters)
- [ ] Target keywords appear in meta titles
- [ ] No duplicate meta titles across the site
- [ ] No truncated meta titles or descriptions

### Headings
- [ ] Every page has exactly one H1 tag
- [ ] H1 contains the primary keyword
- [ ] Heading hierarchy is logical (H1→H2→H3)
- [ ] No heading tags used for styling purposes

### Content Quality
- [ ] Product descriptions are unique (not manufacturer copy)
- [ ] Product descriptions are 150+ words
- [ ] Collection pages have editorial content (800+ words)
- [ ] Blog posts are 1,500+ words with proper structure
- [ ] No thin content pages (under 100 words)

### Images
- [ ] All images have descriptive alt text
- [ ] Image file names are descriptive (not IMG_4523.jpg)
- [ ] Images are appropriately sized
- [ ] Image sitemap is submitted

## Part 3: Structured Data Audit

- [ ] Product schema on all product pages
- [ ] AggregateRating schema (if reviews exist)
- [ ] BreadcrumbList schema site-wide
- [ ] Organization schema on homepage
- [ ] FAQPage schema on pages with FAQ content
- [ ] All schema validates in Google's Rich Results Test
- [ ] Schema data matches visible page content

## Part 4: Internal Linking Audit

- [ ] All products are reachable within 3 clicks from homepage
- [ ] Collections link to subcollections and key products
- [ ] Blog posts link to relevant products and collections
- [ ] No orphan pages (pages with zero internal links)
- [ ] Anchor text is descriptive and varied
- [ ] No broken internal links

## Part 5: Content Gap Audit

- [ ] Competitor content analysis completed
- [ ] Non-brand keyword opportunities identified
- [ ] Blog topic gaps mapped
- [ ] FAQ opportunities identified from Search Console queries
- [ ] Seasonal content calendar planned
